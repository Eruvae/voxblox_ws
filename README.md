# Usage

Clone the repository with submodules
```
git clone --recursive https://github.com/Eruvae/voxblox_ws.git
```

Run the setup script to configure the workspace:
```
cd voxblox_ws
./setup_workspace.sh
```

Build the workspace:
```
catkin build
```

Source the workspace:
```
source devel_release/setup.bash
```
